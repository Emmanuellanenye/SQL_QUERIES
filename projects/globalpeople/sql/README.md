## Project Overview

This project analyzes workforce and employee data for GlobalPeople using SQL to uncover insights related to workforce demographics, employee distribution, and organizational trends. The analysis supports data-driven HR and business decisions by transforming raw employee data into actionable insights.


## Key data fields include:

- Employee identifiers

- Demographic attributes (age, gender, region)

- Employment and organizational details

- Performance-related metrics

## Objectives

- Analyze workforce composition and demographics

- Identify employee distribution across departments or regions

- Support HR and management decision-making using structured SQL queries

- Demonstrate SQL proficiency in real-world workforce analytics

## Key SQL Analysis Areas

## The SQL queries in this project focus on:

- Workforce demographic distribution

- Employee counts and grouping by key attributes

- Organizational insights using aggregations

- Data filtering and transformation using SQL best practices
