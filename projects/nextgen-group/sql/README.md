## NextGen Group — SQL Data Analysis Project
## Project Overview

This project uses SQL to analyze NextGen Group’s donation and engagement data with the goal of uncovering insights into fundraising performance, donor activity, and overall contribution trends. The analysis focuses on aggregating key metrics that support data-driven decision-making for nonprofit and community-focused initiatives.

Using structured SQL queries, the project transforms raw donation records into meaningful summaries that highlight total contributions, activity volume, and performance indicators.

## Key Questions Addressed

- What is the total amount of donations received?

- How does donation activity vary across records?

- What high-level trends can be observed from aggregated donation data?

- How can SQL be used to efficiently summarize and validate contribution datasets?

## SQL Techniques Used

- Aggregate functions (SUM, COUNT)

- Aliasing for readable outputs

- Data validation queries

- Clean, well-documented SQL scripts

## Tools & Technologies

- PostgreSQL

- SQL

- pgAdmin 4

## Business Value

This analysis helps NextGen Group:

- Quickly assess overall fundraising performance

- Validate donation records through aggregation

- Support reporting and transparency efforts

- Enable data-driven insights for strategic planning
